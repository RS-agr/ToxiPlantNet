# ToxiPlantNet: A UAV-Based Deep Learning Framework for Toxic Plant DetectionntNet
**[🚨 News]** This repository is the official implementation of **ToxiPlantNet**. 
> *The full source code, pre-trained weights, and deployment scripts are currently being cleaned up and will be made publicly available upon the acceptance of the manuscript.*

> ## 📖 About The Project

**ToxiPlantNet** is a pioneering UAV-based spatial analysis framework designed to facilitate a paradigm shift from basic individual-level detection to comprehensive ecological dynamic modeling in natural grasslands. 

Using *Euphorbia fischeriana* as a representative species, this framework integrates high-precision deep learning with quantitative spatial indicators to characterize the fine-scale spatiotemporal evolution of toxic plant populations.

1. **Algorithmic Innovation (TOXI-YOLO):** A high-precision, highly lightweight detector tailored for complex grassland backgrounds. It integrates the **GD-RevCol** backbone, **ToxiFusion-Pyramid (TFP)** neck, and **Dynamic Sparse Head (DS-Head)** to achieve robust identification and sub-meter geolocation.
2. **Ecological Metric Derivation:** Introduces two novel geospatial indicators: the **Spatial Aggregation Index (SAI)** and the **Detectable Density Change (DDC)**, transitioning from bounding boxes to quantitative spatial clustering intensity.
3. **Spatiotemporal Application:** Provides a scalable approach for dynamic risk assessment and precision grazing route optimization to support modern animal husbandry.
